Guia de Licenças de Software

Entenda e escolha a licença ideal para o seu projeto

Encontre a licença perfeita para seu software

Escolher a licença correta é uma decisão importante que define como seu código pode ser usado, modificado e distribuído. Este guia ajuda você a entender as principais licenças e fazer a escolha mais adequada para as suas necessidades.

MIT License Permissiva

Uma licença permissiva simples e curta que exige apenas a preservação do aviso de direitos autorais e isenção de responsabilidade.

Uso Comercial Modificação Distribuição Uso Privado Responsabilidade

Apache License 2.0 Permissiva

Uma licença permissiva que também fornece uma concessão expressa de direitos de patente e proteção contra reivindicações de patentes.

Uso Comercial Modificação Distribuição Patentes Marca Registrada Responsabilidade

BSD 2-Clause Permissiva

Uma licença permissiva semelhante à MIT, mas com uma cláusula específica que proíbe o uso do nome do titular dos direitos autorais para promoção.

Uso Comercial Modificação Distribuição Uso Privado Responsabilidade

BSD 3-Clause Permissiva

Semelhante à BSD 2-Clause, mas com uma cláusula adicional que proíbe o uso do nome dos colaboradores para endossar produtos derivados.

Uso Comercial Modificação Distribuição Uso Privado Endosso Responsabilidade

GNU GPL v3 Copyleft

Uma licença copyleft forte que exige que qualquer software derivado também seja distribuído sob a mesma licença.

Uso Comercial Modificação Distribuição Patentes Código Fechado Responsabilidade

GNU LGPL Copyleft

Uma licença copyleft mais permissiva que permite a vinculação com software proprietário sem exigir que este seja liberado sob a mesma licença.

Uso Comercial Modificação Distribuição Vinculação Código Modificado Responsabilidade

Mozilla Public License Copyleft

Uma licença copyleft moderada que permite a combinação com código proprietário.

Uso Comercial Modificação Distribuição Uso Privado Responsabilidade Código Modificado

Creative Commons Variadas

Uma família de licenças para obras criativas com diferentes combinações de atribuições, distribuição e modificação.

Atribuição (BY) Compartilhamento (SA) Uso Comercial (NC) Derivação (ND)

Eclipse Public License Copyleft

Uma licença copyleft que permite vinculação com software proprietário e inclui proteções de patente.

Uso Comercial Modificação Distribuição Patentes Código Modificado

Unlicense Domínio Público

Uma licença que dedica o trabalho ao domínio público, renunciando a todos os direitos autorais.

Uso Comercial Modificação Distribuição Uso Privado Responsabilidade

Domínio Público Domínio Público

Obras que não estão protegidas por direitos autorais ou cujos direitos expiraram, permitindo uso irrestrito.

Uso Comercial Modificação Distribuição Uso Irrestrito Responsabilidade

Artistic License Permissiva

Uma licença frequentemente usada para software Perl que permite uso comercial com algumas restrições.

Uso Comercial Modificação Distribuição Integridade Responsabilidade

Assistente de Seleção de Licença

Responda algumas perguntas simples para encontrar a licença mais adequada para o seu projeto:

1. Você quer permitir uso comercial do seu software?

2. Você quer que modificações do seu código também sejam de código aberto?

3. Você se preocupa com patentes de software?

4. Qual é o nível de liberdade que você quer para os usuários?
